## English

### 📖 About

**Eclipse Canticle** is an ambitious Minecraft mod developed entirely by a solo developer, starting from November 7, 2025. This mod aims to provide a comprehensive expansion to Minecraft, introducing entirely new dimensions to explore, unique biomes to discover, powerful bosses to challenge, and countless new items and mechanics to master.

The mod is built on the Fabric modding platform using Yarn mappings, starting with Minecraft 1.20.1 and continuously updating to support newer versions.

## Introduction
Obviously, the deities we so devoutly worship and revere have never truly favored us. They reign high above, regarding all living beings as mere ants—playthings in their palms, trapped beasts in a cage, nothing more than insignificant pawns in an eternal game.

Then came the day when they effortlessly veiled the sky—the sun remained the same sun, yet its light was swallowed in an instant by an invisible hand. Day turned into endless night, warmth gave way to biting cold. It was a "solar eclipse" cast down by the gods, an unending judgment. We were cast into an abyss of despair, left with only two choices: rise up, defeat all the lofty celestials, and reclaim our light; or forever languish in darkness, withering away in hunger, cold, and fear…

In this world, you shall bear the destiny of the "Savior." Traverse twisted alien dimensions, face powerful guardians who tower above mortals, and continually push beyond your limits. Your final trial will be a decisive battle against the indescribable celestial—■■■■, a confrontation that will determine the fate of the world.

Only by defeating this god can you return to the town where your journey began, activate the ancient sacred altar, dispel the eternal shadow shrouding the sun, and let light once again bathe this land—freeing humanity, at last, from being mere playthings of the divine.

### 📦 Installation

#### Requirements
- Minecraft 1.20.1 or higher
- Fabric Loader 0.15.0 or higher
- Fabric API 0.92.0 or higher
- Java 17 or higher

#### Steps
1. Install [Fabric Loader](https://fabricmc.net/use/)
2. Download [Fabric API](https://modrinth.com/mod/fabric-api)
3. Download Eclipse Canticle from [Releases](https://github.com/YunMu-MC/Eclipse-Canticle)
4. Place both mods in your `.minecraft/mods` folder
5. Launch Minecraft with the Fabric profile

### 🛠️ For Developers

#### Building from Source

```bash
# Clone the repository
git clone https://github.com/yourusername/eclipse-canticle.git
cd eclipse-canticle

# Build the mod
./gradlew build

# Run client for testing
./gradlew runClient

# Run server for testing
./gradlew runServer
```

#### Development Setup

```bash
# Setup development workspace
./gradlew setupDecompWorkspace

# Generate sources
./gradlew genSources

# Run data generation
./gradlew runDatagen
```

## 中文

### 📖 关于

**Eclipse Canticle（日蚀圣咏）** 是一个由独立开发者从 2025 年 11 月 7 日开始完全独立开发的雄心勃勃的 Minecraft 模组。本模组旨在为 Minecraft 提供全面的内容扩展，引入全新的维度供玩家探索、独特的群系等待发现、强大的Boss等待挑战，以及无数新物品和游戏机制等待掌握。

本模组基于 Fabric 模组平台开发，使用 Yarn 映射，从 Minecraft 1.20.1 开始，并将持续更新以支持更新的版本。

## 介绍
很显然，我们平日所虔诚供奉、顶礼膜拜的那些神灵，从未真正垂怜过我们。祂们高高在上，视众生为蝼蚁，将我们视作掌中玩物、笼中困兽，不过是一场永恒游戏里微不足道的棋子。

于是那一天，祂们轻易遮蔽了天空——太阳依旧是那个太阳，光芒却在一瞬间被无形之手覆盖。白昼沦为永夜，温暖被冰冷吞噬，那是神灵降下的“日食”，一场没有尽头的审判。我们被抛入绝望的深渊，只剩下两个选择：要么挺身而出，击败所有高高在上的天神，夺回属于我们的光明；要么，永远沉沦于黑暗，在饥寒与恐惧中痛苦地凋零……

而在这个世界中，你将肩负起“救世主”的命运。穿越被扭曲的异界维度，迎战一个个凌驾于凡人之上的强大守护者，不断突破自身的极限。你的终点，是与那不可名状的天神——■■■■，展开决定世界存亡的最终决战。

唯有战胜祂，你才能重返诞生的城镇，激活古老的圣坛，驱散遮蔽日轮的永恒阴影，让光芒重新照耀这片土地——让人类，真正摆脱被神明玩弄的命运。
